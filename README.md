# MemeHub

## 📝 Prerequisites: 

- Node v22 (LTS)
- pnpm

## ℹ️ Getting started:

### Backend:
Run these commands in the `/backend` directory

- Installing dependencies: 
    
    ```shell
    pnpm run dev
    ```

- Starting dev server:
    
    ```shell
    pnpm run dev
    ```

### Frontend
Run these commands in the `/frontend` directory

- Installing dependencies: 
    
    ```shell
    pnpm run dev
    ```

- Starting dev server:
    
    ```shell
    pnpm run dev
    ```
